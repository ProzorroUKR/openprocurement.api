.. _cdb_context:

Context
=======

Дані які зберігаются в межах одного запиту, реалізований на `threading.local()` або аналогах.

Приклади таких даних:

- Бізнесовий час запиту `get_now()`
- Значення `X-Request-ID`
- `get_request()`
- Сесія роботи з БД `get_db_session()`

Важливо: Об'єкти контекст запиту `get_tender()`, `get_bid()`, etc що посилаются на відповідний ключ в `request.validated`, використовуются лише в межах рефакторінгу (для зручності).

В наступних версіях стейт класи та функції проботи з БД мають працювати ЯВНО з цими об'єктами!
