ItemAttribute
=============

Схема
-----

:name:
    рядок, обов’язковий

:unit:
    :ref:`Unit`

:values:
    Список рядків

    Значення мають бути зі списку `requirement.expectedValues`.

:value:
    boolean/int/decimal

    Значення відповіді на цю вимогу. Значення має бути типом, визначеним у полі requirement.dataType.
