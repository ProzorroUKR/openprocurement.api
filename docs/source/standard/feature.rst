Feature
=======

Схема
-----

:code:
    рядок, генерується автоматично

    Код нецінового критерію.

:featureOf:
    рядок, обов’язковий

    Можливі значення:

    * `tenderer` - учасник закупівлі
    * `lot`
    * `item`

:relatedItem:
    рядок

    ID пов’язаного :ref:`item` або :ref:`lot` (якщо значення ``featureOf`` дорівнює ``item`` або ``lot``).

:title:
    рядок, багатомовний, обов’язковий

    Назва критерію.

:description:
    рядок, багатомовний

    Опис критерію.

:enum:
    список

    Список значень:

    :value:
        float, обов’язково

        Значення критерію.

    :title:
        рядок, багатомовний, обов’язковий

        Назва значення.

    :description:
        рядок, багатомовний

        Опис значення.
